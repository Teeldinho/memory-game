# <div align="center">CONCENTRATION (MEMORY) GAME

Concentration, also known as Memory, Matching Pairs, Match Match, Match Up, Pelmanism, Pexeso or simply, Pairs, is a card game in which all of the cards are laid face down on a surface and two cards are flipped face up over each turn. The object of the game is to turn over pairs of matching cards.</div>
